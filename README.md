## 🚀 快速开始

### 本地开发
```bash
# 安装依赖
bundle install

# 本地运行
bundle exec jekyll serve

# 访问 http://localhost:4000
```

### 部署到GitHub Pages
1. 创建GitHub仓库：`smart-contract-security-guide`
2. 推送代码到`main`分支
3. 在仓库设置中启用GitHub Pages
4. 访问：`https://[username].github.io/smart-contract-security-guide`

## 🎨 自定义配置

### 修改网站信息
编辑`_config.yml`：
```yaml
title: "智能合约安全指南"
description: "专业的智能合约安全审计知识、工具教程和案例分析"
url: "https://your-domain.com"  # 你的域名
```

### 添加新文章
在`_posts/`目录创建Markdown文件，命名格式：`YYYY-MM-DD-title.md`

### 添加新页面
在根目录创建Markdown文件，添加YAML头信息：
```yaml
---
layout: page
title: "页面标题"
permalink: /page-url/
---
```
